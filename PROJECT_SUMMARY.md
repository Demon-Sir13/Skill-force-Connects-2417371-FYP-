# SkillForce — Complete Project Summary

## 🎯 Project Overview

SkillForce is a **production-ready, premium SaaS workforce marketplace** built for Nepal. It connects organizations with skilled service providers through a professional hiring workflow — applications, contracts, messaging, and NPR payments via PeriPay.

## Tech Stack
- **Frontend:** React 18 + Vite, Tailwind CSS, Framer Motion, Recharts
- **Backend:** Node.js, Express, MongoDB/Mongoose, Socket.io
- **Auth:** JWT + 2FA OTP, bcrypt, trusted devices
- **Payments:** PeriPay (NPR), subscription plans
- **Upload:** Cloudinary (images), Multer
- **Real-time:** Socket.io for messaging & notifications

## Key Features
- 2-step login (email/password → OTP verification)
- Professional application flow (cover letter, CV, portfolio, salary)
- Contract generation & digital signing
- Real-time messaging (10 free, then premium)
- Admin panel with full user/job/payment management
- Nepal-specific: NPR currency, districts, local categories
- Dark premium SaaS UI with glassmorphism
